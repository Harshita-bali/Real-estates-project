# **App Name**: ApnaAddress

## Core Features:

- Homepage with Property Search: A landing page featuring a search bar to find properties by city, property type, and budget.
- Property Listings: Dedicated pages for residential, commercial, plots/land, and rental properties, each with detailed information and images.
- Property Submission Form: A form for property owners and brokers to submit property details, including property type, location, size, price, and images.
- Contact Page: A contact form with a Google Maps integration to allow users to directly inquire about properties.
- AI-Powered Property Recommendation Tool: LLM analyzes user search history, saved properties and stated preferences and recommends listings, which might be of interest.

## Style Guidelines:

- Primary color: Deep sky blue (#42A5F5) for a trustworthy and calming feel, inspired by clear skies and stability.
- Background color: Very light blue (#E3F2FD), provides a clean and unobtrusive backdrop that enhances focus on the content.
- Accent color: Light indigo (#7986CB), offers a sophisticated contrast to the primary blue, guiding users to key actions and information.
- Body and headline font: 'Inter', a sans-serif font for a modern and readable design.
- Consistent use of simple and modern icons for property types and features to enhance usability.
- A clean, grid-based layout ensuring responsiveness and easy navigation across devices.
- Subtle hover effects and transitions to provide a smooth and engaging user experience.
